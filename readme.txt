== Description ==

This WordPress plugin adds a new block type to the Gutenberg editor that makes it possible to use LiaScript markdown directly in the editor.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/liascript-markdown-block` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
